About the Project

I wanted to make a game that didn't just rely on the command line and instead used a GUI. I settled on creating Solitaire because it is a fun card game to play when alone. To create the game, I used a Python module called pygame to display the cards and make them clickable. Currently the project is a work in progress, with more features still to be added!

How to play

Learn the rules of Solitaire here

Controls

While in a game, if the r key is pressed then a new game will be started

Click on a card to select it and all cards below (They will be highlighted by a yellow border)

Once selected, click on a pile you would like to move the card/s to

Right click to deselect

Draw three cards setting in options menu
